💻 Interactive Bootstrap Dashboard with Validated Form

📘 Activity Description

This project is a responsive Bootstrap Dashboard created using HTML5, CSS3, Bootstrap 5, and JavaScript. It demonstrates responsive layouts, Bootstrap components, modal functionality, and form validation using Bootstrap’s validation classes and JavaScript.

🧩 Features

    - Responsive Bootstrap Navbar with hamburger menu
    - Bootstrap Card component for dashboard content
    - Interactive Bootstrap Modal
    - Validated contact form inside the Modal
    - Form fields with validation feedback
    - Custom Bootstrap theme color using CSS variables
    - Mobile-first responsive design

🧠 What I Learned

    - Using Bootstrap Grid System and responsive breakpoints
    - Creating responsive navigation bars
    - Building Bootstrap Cards and Modals
    - Implementing form validation with `checkValidity()`
    - Using `.needs-validation` and `.was-validated`
    - Displaying `.invalid-feedback` messages
    - Handling events with JavaScript
    - Customizing Bootstrap colors using CSS variables
    - Designing mobile-friendly interfaces
